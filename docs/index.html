TrainLogic: Logic and Type theory

Course Contents

This course of lectures is an introduction to logic and type theory as background material for formal methods.

Topics:

  • First order logic.
  • Set theory.
  • Type theory.
  • Lambda Calculus

The lectures will be complemented by software implementations and programming exercises. The type theory material will be developed using the dependently typed language Idris. Connections with the use of dependent type theory to integrate computations and proofs will be explored.

The code for this course, including the live code in lectures is available in the repository (the octocat on the top right corner also links to this).

Aims, Background etc

The main aim of this course is to give a self-contained exposition of the (topological) foundations of mathematics and computation known as Homotopy Type Theory. These are foundations that are close to real-life mathematics, allow an integration of programs and proofs, and have deep connection with .

Suggested books

  1. Ebbinghaus, Flum and Thomas, Mathematical Logic, Springer-Verlag.
  2. Martin-Lof, Intutionist Type Theory.
  3. Hindley and Seldin Lambda-Calculus and Combinators: An Introduction, Cambridge.
  4. Edwin Brady, Type-Driven Development with Idris, Manning Publications.
  5. Homotopy Type Theory: Univalent Foundations of Mathematics, Institute for Advanced Studies, Princeton 2013; available at http://homotopytypetheory.org/book/.
  6. Srivastava, S. M., A Course on Mathematical Logic, Universitext, Springer-Verlag, 2008.
  7. Manin, Yu. I., A Course in Mathematical Logic for Mathematicians, Second Edition ,Graduate Texts in Mathematics, Springer-Verlag, 2010.
